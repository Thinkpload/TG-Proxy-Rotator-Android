# План функционала v1 — TG Proxy Rotator Android

## Основное
- Список рабочих MTProto-прокси (сервер, порт, страна, пинг)
- Кнопка «Обновить» — заново загрузить и проверить прокси
- Тап по прокси → открывает Telegram с запросом подключения (`tg://proxy?...`)
- Избранное — пометить прокси, которые стабильно работают

## Автоматика
- Фоновая проверка каждые N часов (настраивается: 1/3/6/12ч)
- Уведомление «Найден свежий прокси — нажми, чтобы применить»
- Автоудаление мёртвых прокси из списка

## Настройки
- Источники прокси (вкл/выкл каждый, добавить свой URL)
- Фильтр по странам (например, только NL/DE/FI)
- Макс. пинг (например, скрыть всё >500мс)
- Тёмная/светлая тема

## Дополнительно (позже)
- История применённых прокси
- Экспорт/импорт списка избранного
- Виджет на главный экран с кнопкой «следующий прокси»
- Статистика: сколько раз какой прокси работал

## Чего НЕ будет (технически невозможно без root)
- Полностью автоматическая смена без тапа — Telegram всегда показывает диалог подтверждения
- Изменение прокси, пока Telegram закрыт
